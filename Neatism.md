# Neatism

### File type supported
* Images
* Documents
* Archives
* Applications
* Torrents
* Fonts
* Videos
* Music
* DB Files
* Game Files
* Web Files

### Usage

* Terminal
  * Copy Neatism.sh in the desired folder and run.
* Cron Job
  * Copy Neatism.sh in the desired folder.
  * ex - 18 23 * * * cd /Users/gaurav/Downloads && ./Neatism.sh >/dev/null 2>&1
